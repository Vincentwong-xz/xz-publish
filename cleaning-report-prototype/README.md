# 清扫报告 - 车辆轨迹结合 Web 原型

基于飞书 PRD《清扫报告-车辆轨迹结合-Web》构建的交互式产品原型。

## 在线预览

直接用浏览器打开 `index.html` 即可运行（需要联网加载 Leaflet 和 OpenStreetMap 瓦片）。

## 功能说明

- **地图轨迹**：使用 Leaflet + OpenStreetMap 展示车辆全天清扫轨迹
- **清扫模式着色**：8 种清扫模式用不同颜色/线型区分，支持图例筛选
- **异常事件标注**：长停、速度异常、绕行、漏扫、重复清扫
- **轨迹回放**：播放/暂停、倍速、进度拖拽、异常锚点跳转
- **左侧面板**：日期/车辆筛选、班次任务时间线
- **底部面板**：异常列表、车辆状态、清扫统计卡片

## 技术栈

- Leaflet 1.9.4
- OpenStreetMap
- 原生 HTML / CSS / JavaScript（单文件）

## 数据说明

所有数据均为 Mock 数据，仅用于产品演示。默认中心坐标为广州生物岛附近。
