# Cloud Helper 原型

无人环卫车辆云端协助平台原型设计。

## 页面

- **index.html** — 总览首页（监控网格 + 车辆列表）
- **vehicle-detail.html** — 车辆详情（含路沿/路基俯视图）
- **personal-stats.html** — 个人接单统计（时间/车辆/Hint 筛选）
